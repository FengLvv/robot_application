# WoZ方法分析

## 什么是WoZ方法

**Wizard-of-Oz (WoZ)** 方法是一种模拟系统交互的技术，通过人工模拟系统行为，为多模态数据的低成本、高效收集提供可行的解决方案。

## WoZ方法的优势

- ✅ 成本较低
- ✅ 更具扩展性
- ✅ 适用于原型评估

## WoZ方法的局限性

通过对WoZ在教育领域的文献研究，总结出以下局限性：

### 1. 操作负担重
操作者需要实时响应用户行为，认知负荷高

### 2. 可能被识破
用户可能察觉到系统并非真正的AI，影响实验效度

### 3. 灵活性不足
不能快速切换技术方案来灵活应变

### 4. 机器人选择受限
研究人员只能在现有集成好的机器人中挑选，存在以下问题：
- 无法自由改变机器人外观
- 无法做出集成系统设定以外的行为
- 无法很好地对未来设计进行验证

## VR技术的优势

考虑将VR技术融入WoZ方法，因为VR具备：

| 优势 | 说明 |
|------|------|
| 操作自由 | 可自定义环境和交互 |
| 沉浸感强 | 提高实验真实感 |
| 操作负担较低 | 相比实体机器人操作更简便 |
| 成本较低 | 无需实体机器人硬件 |

## 研究方向

通过实验探讨**更高拟真度的WoZ方法**对用户评估人-具身智能交互体验的影响，为优化WoZ方法提供一种方法思路。
