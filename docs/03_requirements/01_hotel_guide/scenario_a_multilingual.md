## 二、核心研究问题 (RQs)

### RQ1: 语言切换策略

> **机器人应该如何识别和切换用户的语言偏好？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ1a | **主动切换** vs **被动响应**哪个更受欢迎？ | 切换发起方 |
| RQ1b | 语言切换的**最佳时机**是什么？ | 切换时机 |
| RQ1c | **混合语言**（如中英混用）是否可接受？ | 混合策略 |
| RQ1d | 语言切换如何影响用户**信任度**？ | 信任影响 |

#### 预期发现

- 假设H1a：用户偏好机器人主动识别并切换语言
- 假设H1b：在用户表现出理解困难时切换最自然
- 假设H1c：关键信息用母语、辅助信息可混合

### RQ2: 理解确认策略

> **机器人如何确认用户理解了引导信息？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ2a | **显式确认**（"您理解了吗？"）vs **隐式确认**（观察行为）？ | 确认方式 |
| RQ2b | 确认频率如何影响用户体验？ | 确认频率 |
| RQ2c | 用户未理解时的**重述策略**？ | 重述方法 |

#### 预期发现

- 假设H2a：隐式确认（观察用户行为）更受欢迎
- 假设H2b：过度确认会降低用户满意度
- 假设H2c：简化+视觉辅助比单纯重复更有效

### RQ3: 文化适应行为

> **机器人应该如何调整交互行为以适应不同文化背景？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ3a | **交互距离**应该如何调整？ | 空间行为 |
| RQ3b | **语速和音量**应该如何调整？ | 语音行为 |
| RQ3c | **礼节行为**（鞠躬/握手/点头）如何选择？ | 社交礼节 |

### 3.1 实验设置

```
┌─────────────────────────────────────────────────────────────────┐

│       (被试扮演)        (Wizard控制)                           │
│                                                                 │
│  [多语言标识]  [信息屏]  [引导线]                               │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 实验条件 (2×2 设计)

| | **主动语言切换** | **被动语言切换** |
|---|---|---|
| **显式理解确认** | 机器人主动切换 + 直接询问理解 | 等用户请求 + 直接询问理解 |
| **隐式理解确认** | 机器人主动切换 + 观察行为确认 | 等用户请求 + 观察行为确认 |

### 3.3 被试设计

| 组别 | 被试特征 | 模拟方式 |
|------|----------|----------|
| **A组** | 英语母语者 | 被试用英语交互 |
| **B组** | 非英语外国人 | 被试模拟有限英语能力 |
| **C组** | 中文母语者 | 对照组 |

### 3.4 测量指标

| 类型 | 指标 | 测量方法 |
|------|------|----------|
| **行为** | 任务完成率、完成时间、求助次数 | 系统日志 |
| **主观** | 满意度、舒适度、被尊重感 | 问卷 (Likert) |
| **认知** | 信息理解度、方向感 | 问卷 + 任务测试 |
| **偏好** | 语言策略偏好、交互方式偏好 | 选择题 + 访谈 |

### 3.5 实验流程

```
1. 知情同意 + 背景问卷 (5min)
   - 语言能力自评
   - 跨文化经验
        ↓
2. VR环境熟悉 (5min)
        ↓
3. 角色说明 (3min)
   - 扮演外国游客
   - 任务目标说明
        ↓
4. 实验任务 (每条件8min × 4条件 = 32min)
   - 任务1: 询问餐厅位置
   - 任务2: 请求带路去电梯
   - 任务3: 复杂问询（多步骤）
        ↓
5. 后测问卷 (10min)
        ↓
6. 半结构化访谈 (15min)
```
