## 一、场景概述

| 项目 | 内容 |
|------|------|
| **场景名称** | 整理卧室 (Tidying Bedroom) |
| **核心指令** | "帮我收拾一下卧室" |
| **涉及的边界** | 空间边界（能进卧室吗？）、物品边界（能碰私人物品吗？） |
| **涉及的歧义** | 程度歧义（收拾到什么程度？）、范围歧义（收拾哪些东西？） |

## 二、核心研究问题 (RQs)

### RQ1: 服务边界协商

> **用户如何与家用机器人协商私密空间的服务边界？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ1a | 用户倾向于**主动设定**还是**被动响应**边界？ | 协商发起方 |
| RQ1b | **边界明确程度**如何影响用户对机器人的信任？ | 信任建立 |
| RQ1c | 用户对不同类型边界（空间/物品/信息）的**敏感度差异**？ | 边界类型 |
| RQ1d | 边界协商的**时机**（任务前/任务中）如何影响用户体验？ | 协商时机 |

#### 预期发现

- 假设H1a：用户更倾向于机器人主动询问边界，而非自己主动设定
- 假设H1b：明确的边界协商会提高用户信任度
- 假设H1c：物品边界（私人物品）比空间边界（房间区域）更敏感

#### 测量

| 变量 | 测量方式 |
|------|----------|
| 边界设定行为 | 行为日志（主动/被动、时机、内容） |
| 信任度 | 信任量表（如 Schaefer's Trust in Automation） |
| 边界敏感度 | 自定义量表 + 访谈 |

---

### RQ2: 行为级歧义

> **用户对"收拾卧室"这类模糊指令的期望差异有多大？如何消解？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ2a | 用户对"收拾"的**程度期望**差异有多大？ | 程度歧义 |
| RQ2b | 用户对"收拾"的**范围期望**差异有多大？ | 范围歧义 |
| RQ2c | 哪些**个人因素**影响用户的期望？ | 个体差异 |
| RQ2d | 机器人**澄清策略**如何影响期望匹配度？ | 消歧方法 |

#### 歧义维度详解

```
"帮我收拾一下卧室"
        │
        ├── 程度歧义 ──────────────────────────────────────┐
        │   • 轻度：把明显杂物归位                          │
        │   • 中度：整理表面 + 叠衣服                       │
        │   • 重度：深度清洁 + 整理抽屉                     │
        │                                                  │
        ├── 范围歧义 ──────────────────────────────────────┤
        │   • 地面：散落的衣物、杂物                        │
        │   • 床铺：被子、枕头                              │
        │   • 书桌：文件、杂物                              │
        │   • 衣柜：内部整理                                │
        │   • 床头柜：私人物品                              │
        │                                                  │
        └── 物品歧义 ──────────────────────────────────────┘
            • 普通物品：衣服、书籍
            • 私人物品：日记、照片
            • 贵重物品：首饰、电子设备
```

#### 预期发现

- 假设H2a：用户对"收拾程度"的期望存在显著个体差异
- 假设H2b：整洁度偏好、性格特质会影响期望
- 假设H2c：渐进式澄清比一次性询问更能匹配用户期望

#### 测量

| 变量 | 测量方式 |
|------|----------|
| 期望程度 | 任务前问卷（预期）+ 任务后问卷（实际） |
| 期望匹配度 | 预期-实际差异分数 |
| 个人因素 | 整洁度偏好量表、Big Five人格量表（简版） |
| 满意度 | 任务满意度量表 |

---

### RQ3: 协商交互设计

> **什么样的人机协商方式能最有效地消解歧义、建立边界、提升用户体验？**

#### 子问题

| 编号 | 子问题 | 研究焦点 |
|------|--------|----------|
| RQ3a | **一次性询问** vs **渐进式确认**哪个更有效？ | 协商节奏 |
| RQ3b | **视觉展示** vs **纯语言描述**哪个更清晰？ | 协商模态 |
| RQ3c | 如何平衡**用户控制感**与**交互效率**？ | 体验权衡 |
| RQ3d | 协商方式如何影响用户对机器人的**能力感知**？ | 能力归因 |

#### 协商策略对比

```
策略A: 一次性询问                    策略B: 渐进式确认
─────────────────                   ─────────────────
🤖 "您希望我：                       🤖 "我先把地上的衣服收好？"
   1. 简单整理表面                   👤 "好的"
   2. 整理+叠衣服                    🤖 "需要整理书桌吗？"
   3. 深度清洁全部                   👤 "书桌不用"
   请选择"                           🤖 "床头柜呢？"
👤 "2"                               👤 "床头柜别碰"

优点：高效、明确                     优点：灵活、渐进
缺点：选项可能不匹配                 缺点：耗时、打断多
```

#### 预期发现

- 假设H3a：渐进式确认在高歧义任务中更受欢迎
- 假设H3b：视觉展示（如指向物品）能提高理解准确度
- 假设H3c：过多协商会降低效率感但提高控制感

#### 测量

| 变量 | 测量方式 |
|------|----------|
| 协商效率 | 协商轮次、任务完成时间 |
| 控制感 | 控制感量表（Perceived Control） |
| 能力感知 | 机器人能力感知量表 |
| 整体体验 | UEQ（用户体验问卷）或 NASA-TLX |

## 三、VR-WoZ 实验设计

### 3.1 实验设置

```
┌─────────────────────────────────────────────────────┐
│                   VR 卧室场景                        │
│  ┌─────┐    ┌─────┐    ┌─────────────────────┐     │
│  │ 床  │    │衣柜 │    │      书桌           │     │
│  │     │    │     │    │  [杂物、私人物品]    │     │
│  └─────┘    └─────┘    └─────────────────────┘     │
│                                                     │
│  [散落的衣物]  [地上的杂物]  [床头柜上的物品]        │
│                                                     │
│              🤖 机器人                              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 3.2 实验条件 (2×2 设计)

| | **低歧义指令** | **高歧义指令** |
|---|---|---|
| **无边界协商** | "把地上的衣服放到衣柜里" | "帮我收拾一下卧室" |
| **有边界协商** | 同上 + 机器人询问边界 | 同上 + 机器人询问边界 |

### 3.3 Wizard 操作

Wizard（研究者）控制机器人的：
1. **移动和动作**：根据预设脚本或实时判断
2. **语音回应**：选择预设回复或自由输入
3. **协商策略**：
   - 询问程度："您希望我收拾到什么程度？"
   - 询问范围："需要整理书桌上的物品吗？"
   - 询问边界："我可以进入衣柜整理吗？"

### 3.4 测量指标

| 类型 | 指标 | 测量方法 |
|------|------|----------|
| **行为** | 协商次数、任务完成时间 | 系统日志 |
| **主观** | 满意度、信任度、控制感 | 问卷 (Likert) |
| **认知** | 期望匹配度、意外程度 | 问卷 + 访谈 |
| **偏好** | 边界设定偏好 | 选择题 + 访谈 |

### 3.5 实验流程

```
1. 知情同意 + 背景问卷 (5min)
        ↓
2. VR环境熟悉 (5min)
        ↓
3. 实验任务 (每条件10min × 4条件 = 40min)
   - 用户发出指令
   - 机器人执行/协商
   - 用户确认/调整
        ↓
4. 后测问卷 (10min)
        ↓
5. 半结构化访谈 (15min)
```
